# Stocks Viewer

A modern web application for viewing, tracking, and analyzing stock prices and market trends.

## Features (Planned)
- Real-time stock price lookup
- Visual chart representation of historical stock data
- Personalized watchlist
- Latest market news and updates

## Getting Started

### Prerequisites
- Node.js (version 18 or higher recommended)
- Git

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nathee-rua/Stocks-viewer.git
   cd Stocks-viewer
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```

## Tech Stack
- HTML5 / CSS3 / JavaScript
- Stock Market APIs
