## 🔑 ขั้นตอนที่ 1: การสมัครสมาชิกและการเข้าสู่ระบบ (Authentication)

Stocks Viewer ใช้ระบบล็อกอินแบบ **Account ID / Username** ที่รวดเร็วและปลอดภัย โดยยกเว้นขั้นตอนการยืนยันอีเมลเพื่อความสะดวกรวดเร็วในการเข้าใช้งาน

### 1.1 วิธีสมัครสมาชิกใหม่ (Sign Up)
1. เปิดหน้าเว็บแอปพลิเคชันและไปที่หน้าล็อกอิน (หากยังไม่ได้เข้าสู่ระบบ ระบบจะนำทางมาที่หน้า `/login` อัตโนมัติ)
2. คลิกปุ่ม **"Sign Up"** ที่อยู่บริเวณด้านล่างของกล่องล็อกอิน เพื่อสลับเป็นโหมดสร้างบัญชีใหม่
3. กรอกข้อมูลดังนี้:
   * **Full Name**: ชื่อจริงของคุณ (เช่น *John Doe*)
   * **Account ID / Username**: ชื่อผู้ใช้ที่คุณต้องการ (เช่น *trader_01*) โดยใช้เฉพาะตัวอักษรภาษาอังกฤษ ตัวเลข หรือขีดล่าง
   * **Password**: รหัสผ่านความยาวไม่ต่ำกว่า 6 ตัวอักษร
4. คลิกปุ่ม **"Create Account"**
5. **เข้าสู่ระบบทันที**: ระบบฐานข้อมูลจะข้ามขั้นตอนยืนยันอีเมลและเปิดใช้งานบัญชีของคุณทันที โดยแอปพลิเคชันจะทำการล็อกอินและเปลี่ยนเส้นทางไปยังหน้าหลัก (Dashboard) โดยอัตโนมัติ

### 1.2 วิธีเข้าสู่ระบบเดิม (Sign In)
1. กรอก **Account ID / Username** ของคุณในช่องไอดี
2. กรอกรหัสผ่านของคุณในช่อง **Password**
3. คลิกปุ่ม **"Sign In"** เพื่อเริ่มต้นการใช้งานพอร์ตการลงทุนส่วนตัวของคุณ

## ⚙️ ขั้นตอนที่ 2: การตั้งค่าระบบและกุญแจ API (System Settings & API Configuration)

การเชื่อมต่อข้อมูลราคาหุ้นเรียลไทม์ และระบบปัญญาประดิษฐ์วิเคราะห์หุ้น (AI Analyzer) ทำงานร่วมกับ API คีย์ที่คุณเลือกกำหนดค่าได้ผ่านหน้าการตั้งค่า หรือผ่านไฟล์สภาพแวดล้อมระบบ (.env)

### 2.1 วิธีเข้าสู่เมนูตั้งค่า
1. สังเกตแถบเมนูด้านซ้าย (Sidebar)
2. คลิกเลือกเมนู **"Settings"** (ไอคอนฟันเฟือง)

### 2.2 การตั้งค่าข้อมูลหุ้นและการจัดการเมื่อไม่มี API Key (Finnhub API)
แอปพลิเคชันหลักของระบบราคาและข่าวอิงจาก API ของ Finnhub:
* **การสมัครรับ API Key**: เข้าไปสมัครสมาชิกที่ [Finnhub.io](https://finnhub.io/) และคัดลอกคีย์จากหน้าแดชบอร์ดส่วนตัวมาใส่ในไฟล์ `.env` ที่บรรทัด `FINNHUB_API_KEY=...`
* **กรณีที่ไม่มี API Key (หรือใช้เพื่อการทดสอบ)**:
  > [!NOTE]
  > **หากยังไม่มีคีย์ของ Finnhub หรือยังไม่ได้ตั้งค่า**: ระบบของ Stocks Viewer ได้รับการออกแบบให้ทำงานในโหมด **Auto-Mock Fallback** โดยอัตโนมัติ โดยจะดึงข้อมูลตัวเลขจำลองที่มีความเสถียร (AAPL, TSLA, AMZN, ฯลฯ) ขึ้นมารันบนชาร์ต Candle/Volume และพรีวิวราคาเรียลไทม์จำลองให้แทนทันที เพื่อช่วยให้นักพัฒนาและผู้ใช้สามารถรันระบบทดสอบระบบเทรดและจำลองการใช้งานทุกส่วนได้สมบูรณ์โดยที่แอปไม่เกิดปัญหาขัดข้องใดๆ

### 2.3 การตั้งค่าเครื่องมือวิเคราะห์หุ้น AI (AI Model Provider)
ที่หน้าจอ **Settings** ใต้หัวข้อ **AI ADVISOR SETTINGS** คุณสามารถเลือกโมเดลที่ต้องการใช้งานและป้อน **API Token Key** เฉพาะรายโมเดลได้ ดังนี้:

#### A. OpenAI (GPT-4o mini)
* **วิธีค้นหา API Key**:
  1. เข้าไปที่เว็บไซด์ [OpenAI Platform](https://platform.openai.com/)
  2. ล็อกอินเข้าใช้งานบัญชีของคุณ
  3. ไปที่เมนู **API Keys** จากแถบเมนูด้านซ้าย
  4. คลิกปุ่ม **"+ Create new secret key"** ตั้งชื่อแล้วคัดลอกรหัส (รหัสจะปรากฏครั้งเดียวเท่านั้น)
* **การนำไปใช้**: เลือก Provider เป็น **OpenAI** ในเว็บแอปแล้วนำรหัสที่ได้วางลงในช่อง **API Token Key** จากนั้นกด Save

#### B. Google Gemini (2.5 Flash)
* **วิธีค้นหา API Key**:
  1. เข้าไปที่เว็บไซด์ [Google AI Studio](https://aistudio.google.com/)
  2. ล็อกอินด้วยบัญชี Google ของคุณ
  3. คลิกปุ่ม **"Get API key"** ที่อยู่บริเวณมุมซ้ายบน
  4. คลิก **"Create API key"** แล้วเลือกรันคีย์ผ่านโปรเจกต์ของคุณ จากนั้นคัดลอกรหัส API
* **การนำไปใช้**: เลือก Provider เป็น **Google Gemini** นำคีย์วางในช่อง **API Token Key** แล้วกด Save

#### C. xAI Grok
* **วิธีค้นหา API Key**:
  1. เข้าไปที่เว็บไซด์ [xAI Console](https://console.x.ai/)
  2. สมัครบัญชีนักพัฒนาและเข้าสู่ระบบคอนโซล
  3. คลิกแท็บเมนู **API Keys** จากแดชบอร์ด
  4. คลิกปุ่ม **"Create API Key"** ตั้งชื่อสิทธิ์ตามต้องการแล้วคัดลอกรหัส (รหัสขึ้นต้นด้วย `xai-`)
* **การนำไปใช้**: เลือก Provider เป็น **xAI Grok** นำคีย์วางในช่อง **API Token Key** แล้วกด Save

#### D. Ollama (โมเดล Local ประมวลผลในเครื่องฟรี)
* **วิธีใช้งาน**:
  1. ดาวน์โหลดโปรแกรม [Ollama](https://ollama.com/) ติดตั้งลงในคอมพิวเตอร์ของคุณ
  2. เปิดโปรแกรม Command Prompt หรือ Terminal แล้วสั่งรันโมเดลด้วยคำสั่ง `ollama run llama3`
  3. เมื่อเลือก **Ollama (Localhost - llama3)** ในเว็บแอปพลิเคชัน ระบบจะเชื่อมประมวลผลผ่าน local API ที่เครื่องของคุณทันที **โดยไม่จำเป็นต้องกรอก API Token Key** ใดๆ ทั้งสิ้น

หลังจากตั้งค่าเรียบร้อยแล้ว ให้คลิกปุ่ม **"Save Configurations"** เพื่อบันทึกข้อมูลเข้าโปรไฟล์ผู้ใช้ทันที

## 💼 ขั้นตอนที่ 3: การจัดการพอร์ตการลงทุนและรายการเฝ้าดู (Portfolio & Watchlist)

### 3.1 การเพิ่มหุ้นเข้าสู่รายการเฝ้าดู (Watchlist)
1. ที่หน้าแดชบอร์ดหลัก (Dashboard) สังเกตวิดเจ็ต **"My Watchlist"** ที่อยู่ด้านขวามือ
2. คลิกปุ่ม **"+ Add Stock"** เพื่อเปิดรายชื่อหุ้นที่มีให้เลือก
3. คลิกเลือกหุ้นที่คุณต้องการเฝ้าดู (เช่น *AAPL, TSLA, AMZN*)
4. หุ้นที่เลือกจะปรากฏบน Watchlist ทันที พร้อมแสดงกราฟเส้นราคาย่อ (Sparkline), ราคาล่าสุด และเปอร์เซ็นต์การเปลี่ยนแปลงของวันนี้

### 3.2 การบันทึกประวัติการซื้อขายจริง (Transaction Log & Portfolio Page)
ระบบช่วยคำนวณมูลค่าพอร์ตการลงทุนของคุณจากธุรกรรมจริงที่บันทึก
1. ไปที่เมนู **"Portfolio"** (ไอคอนกระเป๋าเงิน) จากแถบด้านซ้าย
2. คลิกปุ่ม **"+ Add Transaction"** ด้านบนเพื่อเปิดป็อปอัปบันทึกธุรกรรม
3. ระบุรายละเอียดดังนี้:
   * **Symbol**: ตัวย่อหุ้น (เช่น *AAPL*)
   * **Type**: เลือกว่าเป็นรายการซื้อ (BUY) หรือขาย (SELL)
   * **Quantity**: จำนวนหุ้น
   * **Price**: ราคาต่อหน่วยที่ทำธุรกรรม
   * **Date**: วันที่ทำรายการ
4. คลิกปุ่ม **"Submit Transaction"**
5. พอร์ตจะอัปเดตมูลค่ารวม อัตราผลตอบแทนสะสม และแสดงบันทึกไว้ในตาราง **"Historical Transaction Log"** ด้านล่าง (คุณสามารถกดไอคอนถังขยะเพื่อลบธุรกรรมที่ป้อนผิดได้ตลอดเวลา)

## 📊 ขั้นตอนที่ 4: การเปรียบเทียบราคาหุ้นเชิงเปรียบเทียบ (Stock Comparison View)

เป็นเครื่องมือเปรียบเทียบประสิทธิภาพอัตราเปอร์เซ็นต์ผลตอบแทนสะสมของกลุ่มหุ้นในช่วงระยะเวลาที่ผ่านมา

1. คลิกเมนู **"Compare"** (ไอคอนกราฟแท่ง) ในแถบเมนูด้านซ้าย
2. คลิกปุ่ม **"+ Add Stock to Compare"** และเลือกตัวย่อหุ้นที่ต้องการเปรียบเทียบ (เปรียบเทียบได้สูงสุด 4 หุ้นพร้อมกัน)
3. ระบบจะเรนเดอร์กราฟเส้นทับซ้อนกัน โดยแปลงราคาทุกหุ้นให้เป็น **% Change** เพื่อให้คุณเปรียบเทียบได้ทันทีว่าหุ้นตัวใดให้ผลตอบแทนดีที่สุดในช่วงเวลาดังกล่าว
4. เอาเมาส์ชี้ไปที่กราฟเพื่อดูรายละเอียดการเปลี่ยนแปลงราคารายวันของแต่ละบริษัท

## 📈 ขั้นตอนที่ 5: การวิเคราะห์ด้วยกราฟขั้นสูงและการรันระบบทดสอบย้อนหลัง (Interactive Charts & Backtesting)

เมนูนี้เป็นหัวใจสำคัญสำหรับนักวิเคราะห์เทคนิค (Technical Analysis)

### 5.1 การเข้าสู่โหมดวิเคราะห์หุ้นรายตัว
1. คลิกเลือกหุ้นที่ต้องการวิเคราะห์จากหน้า Dashboard หรือ คลิกที่สัญลักษณ์หุ้นใน Watchlist
2. หน้าเพจจะนำทางเข้าสู่ระบบวิเคราะห์หุ้นรายตัวเชิงลึก ซึ่งแสดงรายละเอียดที่สำคัญ

### 5.2 การควบคุมการแสดงผลบนกราฟเทคนิค (Chart Overlays)
กราฟแท่งเทียนใช้เทคโนโลยีของ Lightweight Charts พร้อมฟีเจอร์ระดับเวิลด์คลาส:
* **Volume Bars Overlay**: แถบกราฟปริมาณซื้อขายแบบสีจางอยู่ส่วนล่างของกราฟ
* **ปุ่มตัวเลือกแสดงผล (Overlays)**: ด้านบนและด้านล่างของกราฟจะมีเช็คบ็อกซ์ 3 ตัวเลือก:
  * **Show Fast MA (Purple)**: ติ๊กเลือกเพื่อเปิด/ปิดเส้นค่าเฉลี่ยเคลื่อนที่ระยะสั้น (เส้นสีม่วง)
  * **Show Slow MA (Yellow)**: ติ๊กเลือกเพื่อเปิด/ปิดเส้นค่าเฉลี่ยเคลื่อนที่ระยะยาว (เส้นสีเหลือง)
  * **Show Buy/Sell Arrows**: ติ๊กเลือกเพื่อเปิด/ปิดลูกศรระบุจุดเข้าซื้อ (สีเขียวชี้ขึ้น) และจุดขาย (สีแดงชี้ลง)

### 5.3 วิธีใช้งานระบบทดสอบย้อนหลัง (Moving Average Crossover Backtester)
ระบบจำลองกลยุทธ์การตัดกันของเส้น MA (Dual Moving Average Crossover) ย้อนหลัง 250 วัน
1. ไปที่แผงควบคุม **"Dual Moving Average Crossover Backtest"** ด้านล่างกราฟ
2. ลากแถบปรับตั้งค่าช่วงเวลา (Period):
   * **Fast MA Period**: ระยะวันเฉลี่ยระยะสั้น (ค่ามาตรฐานคือ 12 วัน)
   * **Slow MA Period**: ระยะวันเฉลี่ยระยะยาว (ค่ามาตรฐานคือ 26 วัน)
3. คลิกปุ่ม **"Run Backtest Simulation"**
4. **การคำนวณและแสดงผลลัพธ์**: 
   * เส้น MA ทั้งสองเส้น และลูกศรจำลองการเทรด (BUY/SELL) จะปรากฏขึ้นบนกราฟทันที
   * แผงควบคุมจะแสดงค่าสถิติเชิงปริมาณ (Quantitative Metrics) 6 รายการสำคัญ:
     1. **Total Return (%)**: กำไรหรือขาดทุนสะสมรวมตลอดแผนจำลอง
     2. **Win Rate (%)**: สัดส่วนการเทรดที่ได้กำไรจากจำนวนครั้งการเทรดทั้งหมด
     3. **Total Trades**: จำนวนรอบการซื้อขายที่สิ้นสุดกระบวนการ
     4. **Max Drawdown (%)**: จุดที่พอร์ตขาดทุนลดลงลึกที่สุดจากระดับสูงสุด (ใช้วัดความเสี่ยง)
     5. **Profit Factor**: อัตราส่วนกำไรสะสมหารด้วยยอดขาดทุนสะสม (ค่าที่เกิน 1.0 แปลว่ากลยุทธ์ทำกำไรได้ดี)
     6. **Avg Trade Return (%)**: ผลตอบแทนเฉลี่ยต่อหนึ่งรอบธุรกรรม
   * แท็บบาร์ด้านล่างจะเปิดแท็บพิเศษ **"Backtest"** ขึ้นมาอัตโนมัติ เมื่อคลิกเข้าไปคุณจะพบตาราง **Simulated Trades Log** แสดงรายละเอียดวันเข้าซื้อ, วันขาย, ราคาที่ซื้อ/ขาย และผลกำไรเป็นเปอร์เซ็นต์แยกเป็นรายธุรกรรมอย่างละเอียด

## 🤖 ขั้นตอนที่ 6: การใช้ระบบวิเคราะห์หุ้นด้วย AI (AI Portfolio Advisor)

แอปพลิเคชันมาพร้อมกับสมองกลอัจฉริยะวิเคราะห์โครงสร้างข้อมูลตลาด

1. ในหน้าหุ้นรายตัว เลื่อนไปที่แท็บบาร์ด้านล่างและคลิกแท็บ **"AI Advisor"**
2. คลิกปุ่ม **"Run AI Summary Crossovers"** หรือ **"Generate New Analysis"**
3. ระบบจะส่งข้อมูลราคาปัจจุบัน และการตั้งค่าอินดิเคเตอร์ MA ของคุณไปยังโมเดล AI ที่คุณเลือกไว้ในขั้นตอนตั้งค่า
4. AI จะวิเคราะห์ข้อมูลออกมาเป็นรายงานสรุป 4 ข้อใหญ่ พร้อมระบุคำแนะนำสุดท้ายอย่างชัดเจนว่าควรสะสม (ACCUMULATE), ซื้อ (BUY), ถือครอง (HOLD) หรือขายทำกำไร (SELL)
